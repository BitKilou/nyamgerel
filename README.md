# Nyamgerel Gankhuyag — Official Website

Editorial black-and-white website for **Nyamgerel Gankhuyag**, contortionist and handbalancer with Cirque du Soleil's OVO show.

🌐 **Live site:** [nyamgerel.com](https://nyamgerel.com)

## About

Static HTML/CSS/JS website featuring:
- Editorial black-and-white design with Instrument Serif + Inter typography
- Bilingual support (English default, French toggle)
- Responsive layout (desktop + mobile)
- Photo gallery with lightbox revealing original color
- Booking section for performances, gala events, and brand campaigns
- Direct contact via email, Instagram, and YouTube

## Stack

- Pure HTML5 / CSS3 / vanilla JavaScript — no build step required
- 17 WebP images optimized for web (~1.6 MB total)
- Google Fonts (Instrument Serif, Inter)

## Local development

Just open `index.html` in any browser, or serve the directory with any static server:

```bash
python3 -m http.server 8000
```

Then visit [http://localhost:8000](http://localhost:8000).

## Structure

```
.
├── index.html              # Main page
├── assets/
│   ├── style.css           # Editorial B&W styling
│   ├── app.js              # i18n, gallery, lightbox, animations

│   └── img/                # 17 WebP photos

```
